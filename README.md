# Currículo em HTML

Este projeto é um currículo desenvolvido utilizando HTML semântico, com foco em boas práticas de estruturação e organização de conteúdo.

## 📌 Sobre o projeto

O objetivo deste projeto é apresentar minhas informações profissionais de forma clara, estruturada e acessível, simulando um currículo real em formato web.

## 🚀 Tecnologias utilizadas

- HTML5 (semântico)
